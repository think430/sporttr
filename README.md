# Sporttr

**A GNOME-style, cross-platform fitness and workout tracker.**

Sporttr is a focused, privacy-first workout logging application designed with the **Libadwaita** design philosophy. It provides a clean, distraction-free interface for athletes to manage routines, log sessions, and analyze their physical progress using integrated AI tools.

## ✨ Features

* **GNOME/Libadwaita Aesthetic:** A native-feeling UI with support for Light and Dark modes using the official GNOME color palette and Cantarell typography.
* **Adaptive Layout:** Seamlessly transitions between a mobile-friendly bottom-sheet navigation and a desktop-optimized tab bar.
* **Workout Management:** * Create and organize custom workout routines.
    * Track sessions with a specialized "Workout Flow" interface for logging sets and reps in real-time.
* **AI-Powered Insights:** Connect your own OpenAI (or compatible) API key to get personalized evaluations of your workout performance.
* **Visual Progress:** Monitor your consistency with an activity heatmap and streak tracking.
* **Privacy & Portability:**
    * **Local-First:** Your data stays on your device.
    * **PWA Ready:** Install it as a Progressive Web App for full offline support via Service Workers.
    * **Import/Export:** Easily backup or move your data with JSON export/import functionality.

## 🛠️ Technical Stack

* **Frontend:** Vanilla HTML5, CSS3 (Adwaita variables), and JavaScript.
* **Design:** Inspired by the GNOME desktop environment and Libadwaita.
* **Persistence:** LocalStorage for local-first data management.
* **Connectivity:** Service Worker for offline capabilities (PWA).
* **Intelligence:** Integrated API support for LLM-based workout evaluation.

## 🚀 Getting Started

1.  **Open the App:** Simply open `index.html` in any modern web browser or visit the hosted PWA.
2.  **Define Routines:** Head to the "Routines" tab to build your workout plan.
3.  **Start Training:** Use the "Workout Flow" to log your sets as you complete them.
4.  **Analyze:** Configure your AI settings in the "Settings" tab to unlock session evaluations.
